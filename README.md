## Commandline Usage

``::r <name> <prefs>(a/2/c/h/d/fill)``

``::rr`` rerolls a new random based on the prefs stored from ``::r``

``::prevr`` shows the previous generated random

``::prefs`` (shows previously entered preferences from the ``::r`` command)
